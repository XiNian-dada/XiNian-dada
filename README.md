## About Me

你好，我是 **XiNian_dada**。

主要在写 **Minecraft 服务端插件**，以及一些为自己和别人解决实际问题的小工具。  
不太喜欢“为了练习而练习”的项目，更倾向于 **有需求就写，有问题就修**。

最近在做的事情包括：

- 🧱 Spigot / Paper 插件开发（偏实用、偏稳定）
- 🛠️ 一些轻量化工具和服务
- 🦀 使用 Rust 重写或实现更高性能的工具

不追求炫技，每个项目都尽量做到：
